HH.ru Parser

Парсер вакансий с сайта hh.ru с использованием публичного API. Консольное приложение для сбора и анализа вакансий по ключевым словам.

Описание

Программа выполняет поиск вакансий через API HeadHunter, сохраняет результаты в CSV-файл и выводит статистику. Поддерживает многопоточный парсинг для ускорения сбора данных.

Возможности

· Поиск вакансий по ключевым словам
· Настройка количества страниц для парсинга
· Многопоточная обработка запросов
· Сохранение результатов в CSV
· Вывод статистики (количество вакансий, средняя зарплата)
· Фильтрация по зарплате и регионам

Стек технологий

· C++17 — язык программирования
· CPR (C++ Requests) — библиотека для HTTP-запросов
· RapidJSON — парсинг JSON-ответов
· STL (std::thread, std::async, std::vector, std::string, std::fstream) — многопоточность, контейнеры, работа с файлами
· CMake — система сборки

Алгоритмы

· Многопоточный парсинг с использованием std::async
· Парсинг и валидация JSON-структур
· Фильтрация данных по ключевым словам
· Лексикографическая сортировка результатов
· Подсчет статистических метрик

HH.ru Parser

Парсер вакансий с сайта hh.ru с использованием публичного API. Консольное приложение на C++ для сбора и анализа вакансий по ключевым словам.

---

1. Цель разработки

Создать приложение, демонстрирующее:

· Работу с HTTP-запросами
· Парсинг JSON-данных
· Принципы ООП:
  · Инкапсуляция
  · Наследование
  · Полиморфизм
  · Абстракция

---

2. Общая архитектура

Проект построен на иерархии классов с использованием абстрактного базового класса Parser. Все данные управляются контейнером ParserManager.

Основные компоненты:

Иерархия парсеров

· Parser (абстрактный класс)
· Производные классы:
  · HHParser (парсер hh.ru)

Модель данных

· Vacancy — класс для хранения информации о вакансии:
  · ID, название, компания, зарплата, описание, ссылка

Менеджер парсинга

· ParserManager — управляет:
  · Сбором данных
  · Координацией работы парсеров
  · Сохранением результатов

Сохранение данных

· Экспорт в CSV-формат

---

3. Основной функционал

3.1 Управление поиском

· Ввод ключевого слова для поиска
· Фиксированное количество страниц (3 страницы)

3.2 Парсинг данных

· Отправка GET-запросов к API hh.ru
· Обработка JSON-ответов
· Извлечение полей вакансий

3.3 Отчеты и статистика

· Вывод результатов в консоль
· Сохранение в CSV
· Подсчет статистики:
  · Общее количество вакансий
  · Средняя зарплата

---

4. Требования к реализации

4.1 Обязательные

· Абстрактный класс Parser с чисто виртуальным методом parse()
· Композиция: ParserManager содержит коллекцию парсеров
· Динамическое создание объектов
· Полиморфный вызов методов парсинга
· Работа с JSON через RapidJSON
· HTTP-запросы через CPR

4.2 Дополнительные

· Консольный интерфейс с меню
· Обработка ошибок (таймауты, недоступность API)

---

5. Стек технологий

· C++17 — язык программирования
· CPR (C++ Requests) — HTTP-запросы
· RapidJSON — парсинг JSON
· STL — контейнеры, файлы, алгоритмы
· CMake — система сборки
